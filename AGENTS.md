## Tooling & Shell Usage
- Prefer the bundled bash helpers (`bash -lc`) when invoking shell commands; always set the `workdir` parameter.
- Use `rg`/`rg --files` for searches; fall back only if unavailable.
- Avoid PowerShell-specific commands.
